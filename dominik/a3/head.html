HTML Guide

Common HTML Tags Outside <body>

Tag Definitions Examples
<!DOCTYPE HTML> This tag indicates what language the incoming code is to the browser. The first line of HTML code is always <!DOCTYPE HTML>
<html lang=""> This tag indicates what the default language of the code uses. <html lang="en">
charset This tag indicates what character encoding this code uses. <meta charset="UTF-8">
<title> This tag indicates what is shown on the tab of the website. <title>Assignment 3</title>
Internal CSS Used to change the style of your website using CSS. Found inside the HTML file. <style>body { color: orange;
}</style>
External CSS External CSS is found outside of the HTML and has the same purpose as internal CSS. Found outside the HTML file.
<link rel="stylesheet" href="styles.css">
favicon The favicon is the image displayed on the tab of your site. <link rel="icon" type="image/x-icon" href="favicon.ico">
