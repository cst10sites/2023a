Tag Explanation Example
<p> This tag is used for generic text and is arguably the most common in all of HTML.

"In anarchy, the law falls to the hand of the everyman. We're bound by the judgement of one another."
<h1, h2, h3> The first tag is used for big, iconic text and is often the main header of websites. The other two are smaller headers used for use as subheadings.

"War?"

"This isn't war!"

"War has meaning - a cause to fight for!"
<br> This tag is used for line breaks. It is simply convenient for making things look good.
<ol>, <li> These tags are used to create ordered, ranked lists.
  1. "**** *****."
  2. "Go to college."
<ul>, <li> These tags are used to create unordered, bullet-point-style lists.
  • "We're the Angels of Death."
  • "And your prayers have been answered."
  • "*****!"
<img> This tag is used for images - which are remarkably common online.
<tr, th, td> These tags are used to create tables like the one this text is confined within! Just look around, man.
<hr> This tag is simply a convenient dividing line for when you need some segregation in your website!
