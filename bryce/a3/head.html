Tag Explanation Example
<!DOCTYPE html> This is THE most important tag! It is necessary for your computer to know it's operating a website in the first place. This site alone is proof of it.
<html lang="en"> This tag simply sets the site's default language to English - for features like automatic Google Translate. На этой хоккейной площадке довольно холодно.
charset This tag determines the character set for the site - which is most often UTF 8. Почему 做 彼ら позив هو - هي φούρνος कहिले നിങ്ങൾ 的 안에 ದಿ ठंडा খাদ্য з ອອກ गरम खानु нь ખોરાક?
<title> This is used to set what the tab is titled on the site. This one should be "Assignment 3."
Internal CSS This is CSS that is used within the webpage itself. The only internal CSS I used is to make this table function - as well as sizing the first page's image up.
External CSS This is CSS on an external file that is used to alter the way the site looks and performs. Most of the foundations of this site are built on External CSS.
favicon This simply sets the site's tab icon. I've chosen Fallout: New Vegas' logo as an example.
<link> This is used to connect external sources to your webpage. I used it with both the style sheet and the icon.
<meta> This tag is used to alter the foundations of the website itself. I used it to select the character set.
stylesheet This tag is used to link a style sheet to your webpage which allows for external CSS. It's the reason this page isn't black and white in Arial.
