Tag Definition example
<DOCTYPE html> The DOCTYPE tag tells the computer which code the file is using in this case html.
<html lang="en"> The html lang tag tells the website what language the words need to be written in. This document is written in English since the code is set to English.
<meta charset="UTF-8"> The charset tag tells the website which set of characters to use. The charset recommended for html5 is "UTF-8".
<title></title> The title tag names the name of your tab on your browser. The assignment 3 for the name of the tab is the title.
<style></style> When the <style> tag is used withtin the same html file as the rest of the code it's called internal CSS and with it you can change the appearance of the website ex. the background-color

This text is an h4, using internal CSS I made it light slate grey.
<link rel="stylesheet" href="styles.css"> This code links your code document to another code document called styles.css which allows you do edit the look of the website similar to the internal CSS the external changes the appearance of all of the webpages at once.
<link rel="icon" type="image/x-icon" href="insert name of favicon file here"> The favicon changes the image beside the title on the tab. The heart icon on the tab is my favicon.
