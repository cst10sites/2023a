Common Html Tags inside <head>

Body

Tag Definition Example
<!DOCTYPE html> Indicating that this document is an html file
<html lang="en"> Starts the html code and indicates that it is in english
<meta charset="utf-8"> Indicates that this html code will be using the utf 8 charset
<title></title> Makes the tab title show custom text
<style></style> Lets you type css code inside of the html document
<link rel="stylesheet&# 34; href="style.css"> Links to an external css style sheet
<link rel="icon" type="image/x-icon" href="/images/favicon.ico"> Links a favicon, which can change the tabs logo
