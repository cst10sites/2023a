Common HTML Tags outside <body>

Click here for an HTML Guide about Common HTML Tags inside <body>.

Tag Definition Examples
<!DOCTYPE html> Sets the document type to html This was made in HTML.
<html lang="en"> Everything inside the HTML tag is code and in english. This webpage is in English
charset Sets what kind of characters are used in your html. This is made using UTF-8
title Names the tab at the top of your page The title for this page is "Assignment 3"
internal CSS CSS that goes inside your html document. Uses the <style> tag.
<link rel="stylesheet" href="styles.css"> CSS that goes in a separate CSS document. styles.css
<link rel="icon" type="image/x-icon" href="favicon.ico"> Icon that appears next to your title. The favicon for this page is the number "3"
