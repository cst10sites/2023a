Common HTML Tags outside <body>

Tag Definition Examples
<!DOCTYPE html> It tells the computer that's its an HTML file
<html lang="en"> Sets website default language to English This is english.
<meta charset="UTF-8> Tells website what character set is being used.
<title> Defines title of document and shows in the browser tab. Look at the top left in the tab. "Assignment 3"
Internal CSS: <style> body{ background-color: black; } </style> Internal CSS sets a style for one HTML page. For example, the background of this page is used with Internal CSS.
External CSS: <link rel="stylesheet" href="styles.css"> External CSS is used to define the style of many pages. Also, the background of this page.
Favicon< link rel="icon" type="image/x-icon" href="/images/favicon.ico"> </a> Displays an image to the left of the browser tab.
