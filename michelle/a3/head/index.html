Tag Definition Example
<!DOCTYPE html> Sets the file to an HTML file rather than a regular text file (No example)
<html lang="en"> Sets the file's language to English (No example)
<title></title> This is what text is displayed on the top of your tab(head)
<meta charset="(insert)"> This sets the browser to use a specific type of character encoding (No example)
Internal CSS Typically found in the <head>, internal CSS modifies the HTML file through the <style> element (No example)
<link rel="stylesheet" href="../styles.css"> External CSS is a seperate file that is linked to the HTML file and modifies the entire file (No example)
<link rel="icon" type="image/x-icon" href="favicon.ico"> AKA the Favourite Icon, the small image (icon) beside the title in the head See tab head
