Common HTML Tags outside <body>

Click here for examples of HTML tags inside of <body>

Tag Definition Example
<!DOCTYPE html> This code makes a website, a website.
<html lang="en"> This code determines what language the website is in.
<Meta charset="UTF-8"> This code determines what character set to use.
<title></title> This code makes a title for the code.
Internal CSS Internal CSS is located inside the <head> and is a designer for the webpage internally.
External CSS External CSS is located outside of the code on a seperate page for code and is a designer for the webpage.
<link rel="icon" type="image/x-icon" href="name of image document"> This code makes an image appear next to the page title.
