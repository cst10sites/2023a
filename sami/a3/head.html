<!DOCTYPE html> Specifies that the document is HTML N/A

<title></title> For the title, which goes in the tab with the favicon N/A
Internal CSS (<style></style>) CSS that goes inside the HTML file, inside the <head> <head>
<style>

}
</style>
</head>
External CSS (<link rel="stylesheet" href="">) An external CSS file that is accessed with the code in brackets (Same as for Internal CSS but without <head> or <style>)
Favicon (<link rel="icon" type="image/x-icon" href="">) The image/icon that goes in the tab next to the title N/A
