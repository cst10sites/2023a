Tag Definition Examples
<!DOCTYPE html> the <!DOCTYPE html> tag tells the browser what version of HTML to run. n/a
<htmllang="en"> the <html lang="en"> tag tells your browser to run the website in a certain language, in this case: English. this document is written in English
<charset> the <charset> tag tells the browser to use a certain utf character encoding when translating machine code into human-readable text n/a
<title> the <title> tag defines the text encoded in the tab title. The tab title of this assignment is "Assignment 3"
"internal CSS" Internal CSS is a form of CSS where you do it inside of the HTML document. n/a
"External CSS" External CSS is a form of CSS where you create a second document to hold your CSS code and link the two n/a
<favicon> the <favicon> tag defines the image that registers inside of your browser tab, left of the title. you can view the "favicon" at the left of the tab next to the title
