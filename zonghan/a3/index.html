Common HTML Tags Inside <body>

Tag Definition Examples
<a> Used to create a hyperlink which can direct you to another website.

Click here for hyperlink
<br> Used to produce a line break. Useful for skipping a line for certain formats.

John is 20 year old.
he turns 25 in 5 years,
he was 15 years old 5 years ago,
and is still very young.
<h1> Used for headers and ranges from 1-6 (1 biggest, 6 smallest)

This is a <h1> header
<span> Used for styling text.(Ex:Highlighting words)

My favorite color is cyan
<strong> Used for bolding text This word is bolded
<p> Used for creating a paragraph

This is a paragraph
<ol> Used for creating a ordered list.(numbered list)
  1. This is listed as number 1
  2. This is listed as number 2
  3. This is listed as number 3
<ul> Used for creating unordered list.(bullet points)
  • This is a bullet point.
  • This is another bullet point.
<li> Used for representing an item in a list, it must be contained in a parent element: an ordered list(<ol>) or unordered list(<ul>)
  1. Orange
  2. White
  3. Black
  • Blue
  • Red
  • Yellow
<div> Used as a container for HTML elements and is definied as a division or section in an HTML document that can be styled with CSS.
This table or container is being surrounded by a <div>
<img> Used for embeding an image inside a HTML document.
