Tag Definition Examples
<!DOCTYPE html> It is an "information" to the browser about what document type to expect In HTML 5. you start the document off with <!DOCTYPE html>
<html lang="en"> Used to identify the language of text content on the web. Lang="en" means language the document is using is english.
<charset="UTF-8"> charset is an atribute that goes inside the meta tag, it specifies the character encoding for html document In this document the Charset is "UTF-8"
<title> Defines the document's title that is shown in a browser's title bar or a page's tab. As an example the title of this document is assignment 3.
<style>...</style> Used to define a style for a single HTML page. It is defined in <head> section of the HTML page inside the <style> tag. Example: <!
