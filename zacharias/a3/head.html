Common HTML Tags Outside <Body>

Tag Definition Examples
<!DOCTYPE html> How you set the document type. This is a HTML document.
<html lang="en"> How you set the language for the document. This document is in english.
<meta charset="UTF-8"> How you set the charecter set. This document is set in UTF-8.
<title> Sets the title in the head. This document's title is Assigment 3.
<style></style> How to set internal CSS An example would break the code.
<link rel="stylesheet" href="styles.css"> How to set external CSS. All of the background and fonts and colors.
<link rel="icon" type="image/x-icon" href="favicon.ico"> How you set the favicon. This document's favicon is the HTML logo.
